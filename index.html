Base Station
Edge Computing station, works both in case of online and offline can have a battery backup up to 24-36 hours. On any trigger of any sensor - It calls, text messages and Notifies the first responder group. It helps user to connect with the location using call anytime, anywhere.
PIR/Motion Sensor
Motion sensor can be deployed in any corner, detects human intervention with 8-12 m range. Have hooks for easy installation and has a range of over 100 m.
Door Sensor
Door sensor can be deployed on any door or window and will be triggered when door is opened and the contact is lost from the other part. It have hooks for easy installation.
Panic Button
In emergency situations it can trigger help by one click. It can be used under your desk or on your table and can be pressed whenever the user suspects any life threatening situation.
